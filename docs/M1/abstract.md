# 퀴즈기반 학습 앱
> 단국대학교 소프트웨어학과 4분반 A조 단어즈 팀프로젝트 | 2026년 1학기


## 프로젝트 개요
‘퀴즈맞추기 앱’은 학생들(고등학생,대학생)이 학습 과정에서 겪는 집중력 부족, 낮은 성취도, 비효율적인 공부 방법 등의 문제를 해결하기 위해 개발된 프로젝트이다. 본 앱은 퀴즈 생성, 정답 확인, 오답 복습 기능을 통해 사용자가 자신의 취약한 부분을 효과적으로 학습할 수 있도록 지원하는 것은 목적으로 한다. 이를 통해 학생들은 반복 학습과 자기주도적 학습 능력을 강화하여 보다 효율적인 학습 성과를 얻을 수 있다.

## 주요 기능

| 기능 ID | 주요 기능         | 설명                                            | 우선순위 |
| ----- | ------------- | --------------------------------------------- | ---- |
| FR-01 | 퀴즈 생성 기능      | 사용자는 과목 및 시험 범위를 선택하여 해당 범위에 맞는 퀴즈를 생성할 수 있다. | 상    |
| FR-02 | 퀴즈 풀이 및 답안 제출 | 사용자는 생성된 퀴즈를 풀고 정답을 제출할 수 있다.                 | 상    |
| FR-03 | 정답 및 해설 확인    | 사용자는 퀴즈 풀이 후 정답, 오답, 해설을 확인할 수 있다.            | 중    |
| FR-04 | 오답 저장 및 복습    | 시스템은 틀린 문제를 자동 저장하고 사용자가 다시 풀 수 있도록 지원한다.     | 중    |
| FR-05 | 학습 기록 확인      | 사용자는 점수, 정답률 등 자신의 퀴즈 풀이 기록을 확인할 수 있다.        | 중    |
| NFR-01 | 성능    | 시스템은 퀴즈 생성 및 결과 표시를 0.5초 이내에 완료해야 한다.                     | 중    |
| NFR-02 | 보안    | 시스템은 사용자 학습 데이터 및 기록을 안전하게 저장하고 인증된 사용자만 접근할 수 있도록 해야 한다. | 중    |
