# 퀴즈기반 학습 앱
> 단국대학교 소프트웨어학과 4분반 A조 단어즈 팀프로젝트 | 2026년 1학기


## 프로젝트 개요
학생들이 학습 과정에서 부족한 부분을 효과적으로 파악하고, 퀴즈 생성, 정답 확인, 오답 복습 기능을 통해 반복 학습함으로써 자기주도적 학습 능력과 학업 성취도를 향상시킬 수 있도록 지원하는 맞춤형 학습 보조 앱.
