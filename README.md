# 퀴즈기반 학습 앱
> 단국대학교 소프트웨어학과 4분반 A조 단어즈 팀프로젝트 | 2026년 1학기


## 프로젝트 개요
본 프로젝트는 학생들이 학습 과정에서 부족한 부분을 효과적으로 파악하고, 퀴즈 생성, 정답 확인, 오답 복습 기능을 통해 반복 학습함으로써 자기주도적 학습 능력과 학업 성취도를 향상시킬 수 있도록 지원하는 맞춤형 학습 보조 앱을 개발한다. 기존의 단어 암기 방식은 단순 반복 위주로 이루어져 학습 효율이 낮고, 사용자가 어떤 단어를 자주 틀리는지 체계적으로 관리하기 어렵다는 문제가 있다. 또한 개인의 학습 수준에 맞춘 복습 기능이 부족하여 장기 기억으로 이어지지 않는 한계가 있다. 이에 본 앱은 사용자가 단어를 효과적으로 암기하고 복습할 수 있도록 퀴즈, 오답 노트, 반복 학습 기능을 제공하여 학습 효율과 기억 지속성을 향상시키는 것을 목표로 한다. 주요 사용자는 단어 학습이 필요한 중·고등학생 및 자기주도 학습을 하는 일반 사용자이다.

## 주요 기능

1. 퀴즈 생성 기능: 단어 데이터를 기반으로 다양한 형태의 문제를 자동 생성하고 정답 판별 및 피드백을 제공하는 기능
2. 답안 제출 기능: 사용자가 입력한 답안을 채점하고 정답 여부 및 피드백을 제공하는 기능
3. 정답 및 해설 확인 기능: 정답과 해설을 제공하여 사용자가 틀린 이유를 이해하고 학습을 강화할 수 있도록 하는 기능
4. 오답 저장 및 복습 기능: 틀린 문제를 자동 저장하고 오답만을 반복 학습할 수 있도록 지원하는 기능
5. 학습 기록 확인 기능: 학습 결과 데이터를 제공하여 사용자가 자신의 학습 상태를 분석하고 관리할 수 있도록 하는 기능
